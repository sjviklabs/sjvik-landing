Three new directions
for stevenjvik.tech.

Each variant is a complete homepage that addresses every P0 from the audit: email CTA in hero, quantified metrics ribbon (40% / 3,000+ / 0), promoted case studies, primary tech stack surfaced, and reduced-motion safeguarded. They use the real brand-site palette (forest green on near-black) and reuse existing tokens — the implementation pass converts the chosen direction into an Astro HeroX.astro component plus a $variant store entry.

These are distinct from the existing terminal and editorial variants — each occupies a different aesthetic axis (HUD dashboard vs. federal file vs. engineering journal) so adding any of them grows the system rather than duplicating it.
