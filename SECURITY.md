# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any SJVIK Labs project, please report it responsibly.

**Email:** stevenjvik.work@gmail.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to resolve confirmed vulnerabilities promptly.

## Scope

This is a static website with no server-side processing, user authentication, or data storage. The attack surface is limited to the build pipeline and deployment process.
