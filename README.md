# SJVIK Labs Landing Page

Source for [sjvik-labs.stevenjvik.tech](https://sjvik-labs.stevenjvik.tech) — the SJVIK Labs hub.

## Stack

- [Astro](https://astro.build) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com) — utility-first styling
- Hosted on Hostinger via FTP

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build          # Build to dist/
bash scripts/deploy.sh # Build + FTP deploy to Hostinger
```
